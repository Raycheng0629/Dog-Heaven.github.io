第十五章:CSS Box Model 詳細介紹

Box Model 是 CSS 中的一個基本概念，用於描述元素的內容、內邊距、邊框和外邊距的佈局結構。理解 Box Model 對於掌握 CSS 排版和佈局技巧至關重要。

這是一個示例盒子，用來展示 CSS Box Model 的各個組成部分。

Box Model 各部分詳解

Content： 內容是元素的實際內容區域，如文字或圖片。

Padding： 內邊距是內容與邊框之間的空間，增加了元素的可見大小。

Border： 邊框是圍繞內邊距和內容的邊界，可以設置其寬度、樣式和顏色。

Margin： 外邊距是邊框外部的空間，用於控制元素之間的距離。
